# Prompt: Write an algorithm for finding even and odd numbers

**Normal Solution:**
```
Input Integer Num1
Check if 'Num1' is divisible by 2
      If it is then 
          Output " The Number is Even"
      Else
          Output " The Number is Odd"
      EndIf
  ```
